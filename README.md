# CauldronOnlineLibrary
Common, Server, and Console code

2D Zelda-like library for Server logic and generating Server Data.

Common has all shared data between Client and Server along with a WorldClient to use in a different project. Also includes data needed for creating server data in an Editor.
Server has all the logic required to play the game.
Console is to start the server while also having the ability to issue commands as needed.
